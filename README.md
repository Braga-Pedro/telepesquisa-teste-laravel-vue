O sistema de cadastro de empresas faz parte de uma avaliação para cargo de desenvolvedor PHP | Laravel da empresa Telepesquisa. É desenvolvido com `Inertia` utilizando `Breeze` e `PHP 8.2` | `Laravel 12` | `Vue 3.4`.
